Practice Challenges – Beginner Level

## **4️⃣ Speed Checker**

**Scenario:**
You want to check if a car is **speeding**.

**Rules:**

- Speed > 80 → "speeding"
- Speed <= 80 → "safe"

**Given:**

```js
const speed = 90;
```

**Tasks:**

1. Create a boolean `isSpeeding`
2. Log a sentence:

```
The car is going 90 km/h and is speeding.
```
